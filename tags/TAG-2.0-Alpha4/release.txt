2012/08/27 TomatoCart v2.0 alpha2
  TomatoCart v2.0 is in development, please do not use it in production enviroment. This release is an evaluation version 
  for community developers and users who are participating in the development and are providing feedback.
  
  There will be no install routine for evaluation edition, please refer to readme.txt for installation guide.
  
  Features included in this release:
  1. Developed as a flexible and solid ecommerce framework utilizing mvc pattern.
  
  2. Add and override core features without touching the core source code.
  
  3. Integrate the sencha touch into the admin panel to support the pad and mobile device.
  
  4. Improve the template engine as the engine in the wordpress.
  
  5. Perform modifications and override base views in a separate template for the specific device.
  
  6. Migrate the the sytem config file into the local directory so as to let the developers setup the system without touching the core config file.
  
  7. Share the system config file between Backend and Frontend
  
  8. Add the payment, shipping, order totals and service module into the admin panel.
